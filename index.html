GithubNotifier by ctshryock

Little OS X menu-bar app that posts Growl notifications from your Github network activity

Dependencies

Github Notifier uses the following libraries in the form of submodules: - My fork of CocoaREST from Steven Degutis - EMKeychain from ExtendMac - NSWorkspaceHelper from Shaun Harrison and or Enormego

Install

Grab the latest download here: https://github.com/ctshryock/GithubNotifier/downloads If you'd like to compile and run your own release build go to project settings and under "release" configuration, remove scary-robot-development and either leave blank or use your own key
